# 🇨🇳 Viagem China 2026

Planejador da viagem (30/set – 20/out/2026): Beijing → Xi'an → Chengdu → Guilin/Yangshuo → Xangai.

- **Roteiro**: linha do tempo — cidades, voos e trens em sequência
- **Mapa**: todos os pontos do roteiro (Leaflet + Carto Voyager)
- **Calendário**: 21 dias em cards ou tabela de edição em massa
- **Dia a dia**: cronograma horário, um dia por vez
- **Voos / Hotéis / Trens**: reservas e status
- **Atenção**: checklist pré-viagem com progresso
- **Frases**: mandarim com pinyin + modo tela cheia + voz
- **⬇️ Offline**: exporta arquivo HTML único que funciona sem internet no celular

Login básico: `paulo` edita, `amor` visualiza. Dados no Postgres (Neon), autosave.

Ver `DEPLOY.md` para publicar/rodar.
